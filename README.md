# port-scanner
A simple port scanner created in PYTHON3.
<br>
<h3>Usage: </h3>
<code>python3 port-scanner 172.217.161.4 1337</code><br>
<code>python3 port-scanner help</code><br>
<code>python3 port-scanner [IP-Address] [Port]
</code>
